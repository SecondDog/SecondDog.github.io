OC数据结构（二）：栈的用途

从LeetCode的第20题说起

Given a string containing just the characters ‘(‘, ‘)’, ‘{‘, ‘}’, ‘[‘ and ‘]’, determine if the input string is valid.
The brackets must close in the correct order, “()” and “()[]{}” are all valid but “(]” and “([)]” are not.

大概翻译一下：
输入一个只包含{}()[]几种括号的字符串，判断字符串是否合法。合法的条件你懂的，就是括号都能对得上。例（）and (){}[]都合法 ，(] 和 ([)]不合法

show me the code

这道题目用栈来做简直最得心应手了算法的基本描述如下:
遇到左括就入栈，遇到右括号就从栈里pop出一个左括号并判断是否匹配。一直持续这个过程就OK了
