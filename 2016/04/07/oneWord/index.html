另外我目前在用的几个Xcode插件 备忘一下
ZLGotoSandbox
You can quickly enter the iOS simulator Xcode plugin

XcodeColors
XcodeColors allows you to use colors in the Xcode debugging console.It’s desigend to aid in the debugging process.

DXXcodeConsoleUnicodePlugin
Convert unicode string to Human-readable in Xcode 5+ console

DerivedData Exterminator
Button for quickly deleting derived data. Makes Xcode happy.

CleanHeaders
A simple iSort like header sorting and duplicate removal plugin for Xcode

ApportablePlugin
Simple Plugin for work with Apportable supporting Live Coding
